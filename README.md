# symbios-audio

The Bevy-free core of [`bevy_symbios_audio`](https://github.com/TheJanusStream/bevy_symbios_audio):
a deterministic DAG-of-nodes algorithmic audio synth (oscillators, noise,
ADSR, biquad filters, LFOs, mix/gain, gate, chorus, reverb), a sequence
mixdown layer, a pure RIFF/WAVE IEEE-float encoder, and `symbios-genetics`
integration for every config struct.
